# Policy COV-003: Medical Necessity and Coding

## Scope
Services must be medically necessary and supported by a diagnosis code consistent
with the procedure billed.

## Rules
- COV-003.1: A procedure must be paired with a diagnosis code that supports medical
  necessity. A mismatch is denied under denial code D-NMN (not medically necessary).
- COV-003.2: Claims billed with a diagnosis code that is incomplete or invalid are
  denied under denial code D-CODE (invalid or unspecified coding).
- COV-003.3: Experimental or investigational procedures are denied under denial code
  D-EXP (experimental service) regardless of diagnosis.

## Appeals
A D-NMN denial may be appealed with clinical notes establishing medical necessity.
A D-CODE denial may be appealed by resubmitting with corrected, specific coding.
