# Policy COV-002: Prior Authorization Requirements

## Scope
Certain high-cost procedures and imaging require prior authorization before the
service is rendered.

## Rules
- COV-002.1: Advanced imaging (procedure 70551 MRI brain, 74177 CT abdomen) requires
  an approved prior authorization on file before the date of service.
- COV-002.2: A claim for a service in COV-002.1 without prior authorization is denied
  under denial code D-NOAUTH (no prior authorization).
- COV-002.3: Prior authorization is valid for 60 days from the approval date. A service
  performed after expiration is denied under denial code D-AUTHEXP (authorization expired).

## Appeals
A D-NOAUTH denial may be appealed if the provider can show the service was an emergency,
in which case the prior authorization requirement is waived under COV-002.4.
